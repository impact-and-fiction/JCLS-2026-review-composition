# JCLS-2026-review-composition

Code and data for the 2026 JCLS submission on review composition.

There are 5 notebooks in `scripts` that together generate most of the tables and figures in the paper.

- [notebook 1: agreement](scripts/notebook-1-agreement.ipynb)
- [notebook 2: analysing review composition with human annotations](scripts/notebook-2-review_composition_human_annotations.ipynb)
- [notebook 3: creating ground truth for training annotation prediction](scripts/notebook-3-making-ground-truth.ipynb)
- [notebook 4: evaluation models for annotation prediction](scripts/notebook-4-evaluation.ipynb)
- [notebook 5: analysing review composition with predicted annotations](scripts/notebook-5-review_composition_full_annotations.ipynb)
